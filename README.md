# AI Data Analyst — AI-Powered Data Analysis Web Application

> CISC 520 Final Project | Week 1 Prototype

An agentic AI system that lets users perform data analysis through natural language. Users describe what they want — *"Fetch Apple stock prices for the last 100 days and plot a chart"* — and the agent generates Python code, executes it, interprets the results, and presents findings in a clean chat interface.

---

## Architecture

```
┌─────────────────────────────────────────────────────┐
│                   Browser (Frontend)                 │
│   React chat UI · Code renderer · Chart.js plots    │
└───────────────────────┬─────────────────────────────┘
                        │ HTTP POST /api/analyze
┌───────────────────────▼─────────────────────────────┐
│                 FastAPI Backend                      │
│   Prompt builder → Anthropic API → Code parser      │
│   → Python executor (subprocess + timeout)          │
│   → Result interpreter → Structured JSON response   │
└───────────────────────┬─────────────────────────────┘
                        │ HTTPS
              ┌─────────▼──────────┐
              │   Anthropic API    │
              │  claude-sonnet-4   │
              └────────────────────┘
```

**Agentic pattern implemented:** Multi-step reasoning (plan → generate code → execute → interpret) with self-correction on execution errors and iterative refinement via conversation history.

## Quick Start

### Prerequisites
- Python 3.10+
- Node.js 18+
- An [Anthropic API key](https://console.anthropic.com/)

### 1. Clone & configure

```bash
git clone https://github.com/YOUR_USERNAME/ai-data-analyst.git
cd ai-data-analyst
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
```

### 2. Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
# Open http://localhost:5173
```

## Example Prompts

| Scenario | Prompt |
|----------|--------|
| Stock analysis | `Fetch AAPL closing prices for the last 60 days, plot a line chart, and compute mean, median, and standard deviation` |
| Moving average | `Generate 12 months of sample sales data, compute a 3-month rolling average, and plot both series` |
| Statistics | `Create 200 random normal values, show descriptive statistics, and plot a histogram` |
| Comparison | `Compare monthly returns of TSLA vs MSFT over the past year and run a t-test` |
| Self-correction | `Analyze a dataframe column called revenue_total` *(agent will detect the error and retry)* |

## Known Limitations (Week 1 Prototype)

- Code execution runs in a subprocess, not a fully isolated Docker sandbox — recommended for development only
- No file upload support yet (CSV analysis via upload is planned for Week 2)
- Chart rendering is Chart.js spec generated by the LLM; complex matplotlib figures are not yet captured as images
- No authentication or rate limiting on the API endpoints

## Deployment

The frontend can be deployed to **Vercel** or **Netlify** (static). The backend can be deployed to **Railway**, **Render**, or **Fly.io**.

```bash
# Frontend build
cd frontend && npm run build   # outputs dist/

# Backend — set env vars on your host, then:
uvicorn main:app --host 0.0.0.0 --port 8000
```
